<h1>らくらくプロットツール</h1>

<h2>概要</h2>
Easy Plot Tool は、測量座標データをブラウザ上で可視化・編集できるインタラクティブなWebアプリです。<br>
地図上にポイントをプロットし、位置関係や構造を直感的に確認できます。<br>

主に以下の用途を想定しています：<br>
<br>
測量データの簡易可視化<br>
座標データの確認・検証

<h2>デモ</h2>
GitHub Pagesで公開しています：<br>
<br>
👉 https://katsuura-yuhei.github.io/Easy-Plot-Tool/


<h2>主な機能</h2>
座標プロット、視覚的な確認<br>
距離、面積、角度の計算<br>
座標データの読み込み、出力<br>
座標の数学表示（x軸横、y軸縦）、測量表示（x軸縦、y軸横）の切り替え<br>
角度表示の弧度法、小数点の切り替え

<h2>読み込み可能なデータセット</h2>
カンマ区切り、名前、x座標、ｙ座標からなるcsvが読み込めます。

<h4>読み込み可能なファイル例</h4>
name,x_co,y_co<br>
a,-10,1<br>
b,10,120<br>
c,50,80<br>
d,-60,65<br>


<h2>使用技術</h2>
HTML / CSS / JavaScript<br>
D3.js

<h2>特徴・工夫した点</h2>
測量・座標データを「直感的に理解できる形」に変換<br>
GISツールより軽量で、ブラウザのみで動作<br>
クライアントサイドで完結、セキュリティ面も安心
